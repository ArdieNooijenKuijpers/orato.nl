## Project Structure

Main application code is in:

`/orato`

Key folders:

- `orato/app` -> App Router pages, layouts, and site components
- `orato/components` -> reusable UI components
- `orato/public` -> static assets
- `orato/lib` -> utility functions

## Getting Started

### 1. Install dependencies

```bash
cd orato
npm install
```

### 2. Run development server

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

### 3. Build for production

```bash
npm run build
npm run start
```

### 4. Lint

```bash
npm run lint
```
